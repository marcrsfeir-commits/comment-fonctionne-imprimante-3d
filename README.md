# Comment fonctionne une imprimante 3D ?

Un site informatif, en français, qui explique simplement le fonctionnement
des imprimantes 3D — avec des images, une vidéo et des animations interactives.

Projet scolaire (Technologie).

## Contenu

- **Guide illustré** : définition, les 3 étapes (modéliser, trancher, imprimer),
  les matériaux (filament), et les domaines d'application.
- **Simulateur** : un objet se construit couche par couche sur un canvas.
- **L'atelier** : on choisit un objet et une couleur de filament, puis on
  regarde l'impression se faire — sans note, juste pour s'amuser.
- **Vidéo** intégrée et **sources** citées.

## Technologies

HTML, CSS et JavaScript (sans aucune bibliothèque externe).
